## 🚀 新增功能

### 1. 全局命令面板 ⭐

**快捷键：** `Ctrl+P`

快速搜索和执行命令，无需鼠标操作。

**功能：**
- 🔍 搜索所有连接（按名称、主机、类型）
- ⚡ 执行常用命令（新建连接、切换视图、打开设置等）
- 🕐 显示最近使用的连接（最多 10 个）
- 📂 命令按类别分组显示

**使用方法：**
1. 按 `Ctrl+P` 打开命令面板
2. 输入关键词搜索（支持模糊匹配）
3. 使用 ↑↓ 键选择，按 Enter 执行
4. 按 Esc 关闭

**示例：**
- 输入 "mysql" → 快速找到所有 MySQL 连接
- 输入 "new" → 显示所有"新建"相关命令
- 输入 "theme" → 切换主题

---

### 2. 工作区状态持久化 💾

**自动保存：** 每次状态变化后 500ms 自动保存

应用关闭后自动保存工作状态，下次启动时恢复。

**保存内容：**
- ✅ 打开的所有标签页
- ✅ 当前活动的标签页
- ✅ 侧边栏折叠状态和宽度
- ✅ 底部面板折叠状态、高度和活动标签

**使用方法：**
- 无需任何操作，自动保存和恢复
- 关闭应用后，下次启动自动恢复上次的工作状态
- 如果连接失效，会自动跳过该标签页

**注意：**
- 不保存查询编辑器的内容（请手动保存重要查询）
- 不保存文件编辑器的内容（请手动保存文件）

### 4. 主题快速切换 🎨

**快捷键：** `Ctrl+K T`（组合键）

快速在深色和浅色主题之间切换。

**使用方法：**
1. 按 `Ctrl+K`（进入组合键模式）
2. 1 秒内按 `T`（执行主题切换）
3. 主题自动切换（深色 ↔ 浅色）

**支持的主题：**
- Default Dark（默认深色）
- Default Light（默认浅色）
- Monokai（深色）
- Dracula（深色）
- Nord（深色）

**手动切换：**
- 打开设置 → 主题 → 选择主题

### 5. 连接导入/导出 📦

**状态：** 后端完成，前端 UI 开发中

支持导入和导出连接配置，方便迁移和备份。

**支持格式：**
- ✅ DevForge JSON（完整支持）
- 🔄 Navicat XML（开发中）
- 🔄 Termius JSON（开发中）

**冲突处理策略：**
- **跳过：** 保留现有连接，跳过冲突项
- **覆盖：** 用导入的连接替换现有连接
- **重命名：** 自动为冲突连接添加后缀

**安全特性：**
- 默认不导出密码（安全考虑）
- 导入时可选择是否导入明文密码
- 版本控制，防止格式不兼容

**使用方法（即将推出）：**
1. 打开设置 → 导入/导出
2. 选择导出或导入
3. 选择文件和冲突策略
4. 预览并确认

## 💡 使用技巧

### 1. 命令面板最佳实践

- **快速连接：** 输入连接名称的前几个字母即可快速匹配
- **最近使用：** 最近打开的连接会显示在顶部，方便快速访问
- **命令搜索：** 不记得快捷键？用命令面板搜索功能名称

### 2. 工作区管理

- **多项目工作：** 为不同项目打开不同的标签页，关闭应用后自动保存
- **快速恢复：** 重启应用后，所有标签页和面板状态自动恢复
- **清理状态：** 如需重置工作区，清除浏览器 localStorage 中的 `devforge-workspace-snapshot`

### 3. 快捷键组合

- **组合键：** `Ctrl+K` 是组合键前缀，后续可以跟其他键
- **超时机制：** 按下 `Ctrl+K` 后，1 秒内按第二个键有效
- **取消组合：** 按 Esc 或等待超时自动取消

### 4. 主题切换

- **快速切换：** 使用 `Ctrl+K T` 在深色和浅色之间快速切换
- **多主题：** 在设置中可以选择 5 个内置主题
- **自定义：** 未来版本将支持自定义主题
